# Coin Basecamp - CoinMarketCal Free Version

## 포함 기능
- CoinDesk RSS 뉴스 자동 로드
- CoinDesk / Cointelegraph 기반 중요 알림
- CoinGecko 실시간 티커
- CoinMarketCal 기반 동적 캘린더

## 필요한 환경변수
- OPENAI_API_KEY (선택): 뉴스 한국어 요약 품질 향상
- COINMARKETCAL_API_KEY (선택): 무료 CoinMarketCal API 연동

COINMARKETCAL_API_KEY가 없으면 캘린더에는 예시 일정이 표시됩니다.
무료 플랜도 사용 가능하지만, API 키 발급이 필요합니다.
